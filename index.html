Instalação

Conheça o comando !fr

O comando !fr chama uma overlay personalizada que ajuda você a indicar um canal amigo ou a recompensar quem fez uma /raid para o seu canal! Também indicamos você configurar um comando no seu bot para enviar uma mensagem no chat.

Nós usamos a função de widget personalizado da StreamElements, ou seja, você precisa ter uma conta para poder utilizar a nossa overlay.

Instalação

Para instalar a overlay é bem simples, você precisará apenas seguir estes passos e depois fazer CTRL+C e CTRL+V.

1º - Depois de logado na StreamElements com a sua conta da Twitch, você precisa clicar em Minhas Overlays e criar uma nova overlay em branco, indicamos com a resolução de 1080p.

2º - Clique agora no botão ADD WIDGET, vá na opção STATIC / CUSTOM e por fim em Custom widget.

3º - Este é o último passo para poder começar a fazer o CTRL+C e CTRL + V, clique em OPEN EDITOR e copie e cole os códigos a baixo em seus devidos lugares:

HTML

CSS





JS



FIELDS



DATA



Agora que você copiou e colou tudo certinho na nova overlay, salvar, clicando em SAVE e depois adicionar em seus programa de stream como um navegador. Para adicionar no programa, você precisa clicar no ícone de link na página ou clicar nos três pontinhos e depois em copiar URL.

Comando no chat

Já temos a overlay no StreamElements agora precisamos do comando que irá aparecer no chat. Escolha o bot que você já utiliza e copie:

StreamElements

Nome do comando: !fr

Resposta:
/me Uma indicação da FeveR para você! Queremos que você conheça ${user} que faz live de ${game ${user}}! Clique no link: https://twitch.tv/${user} imGlitch TwitchLit imGlitch TwitchLit

Streamlabs Cloudbot

Nome do comando: !fr

Resposta:
/me Uma indicação da FeveR para você! Queremos que você conheça {touser.name} que faz live de {touser.game}! Clique no link: https://twitch.tv/{touser.name} imGlitch TwitchLit imGlitch TwitchLit

Streamlabs Chatbot

Nome do comando: !fr

Resposta:
/me Uma indicação da FeveR para você! Queremos que você conheça $touser que faz live de $game! Clique no link: https://twitch.tv/$touser imGlitch TwitchLit imGlitch TwitchLit

Nightbot

Nome do comando: !fr

Resposta:
/me $(twitch $(touser) “Uma indicação da FeveR para você! Queremos que você conheça {{displayName}} que faz live de {{game}}! Clique no link: {{url}} imGlitch TwitchLit imGlitch TwitchLit")

Pronto, agora está tudo certinho para você começar a usar a nossa overlay em suas lives!
